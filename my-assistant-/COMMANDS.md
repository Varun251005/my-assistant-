# 🤖 Assistant Commands Cheat Sheet

## ⏱️ TIME & CALENDAR
```
time                    Get current time
date                    Get today's date
remind me [task] at [time]    Set a reminder
my reminders            List all reminders
```

## 🌤️ WEATHER & NEWS
```
weather                 Get current weather
weather [city]          Get weather for city
news                    Get latest news
```

## 🎵 MUSIC & MEDIA
```
play [song]             Play music on YouTube
volume [0-100]          Set system volume
```

## 📁 FILES & SYSTEM
```
list files              List files in current directory
search [filename]       Find a file
disk space              Check disk usage
system info             Get OS and hardware info
cpu usage               Get CPU usage
memory                  Get RAM usage
```

## 🧮 MATH & CONVERSION
```
calculate [expression]  Do math (e.g., "calculate 2+2*5")
convert [value] [unit1] to [unit2]    Convert units
```

### Common Conversions:
```
convert 10 km to miles      → 6.21 miles
convert 50 lbs to kg        → 22.68 kg
convert 0 celsius to fahrenheit    → 32 fahrenheit
```

## ✅ TODO MANAGEMENT
```
add todo [task]         Add a new todo
my todos                List all todos
mark todo [#] done      Mark a todo complete
```

## 🔍 SEARCH & WEB
```
search [topic]          Google search
who is [person]         Wikipedia search
open [app]              Open an application
```

## 🎭 ENTERTAINMENT
```
tell me a joke          Get a random joke
joke                    Alternative: get a joke
[any question]          Ask anything (intelligent responses)
```

## ℹ️ HELP & EXIT
```
help                    Show all commands
exit / quit / stop      Exit the assistant
```

---

## 💡 TIPS

✓ Start with: `help` to see all commands
✓ Use natural language: "what time is it" or "time"
✓ Voice input works if microphone connected
✓ Falls back to text input automatically
✓ Works offline for most commands (except weather/news/web)
✓ Chain multiple commands: get time → check weather → add todo
